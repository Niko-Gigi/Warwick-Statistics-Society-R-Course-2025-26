We will begin by familiarising ourselves with the operations in R, where

“an operator is a character, or characters, that determine what action is to be performed or considered”,

according to the BBC Bitesize website. These will form the base building blocks of everything we will do in the future.

We will start with arithmetic operators (\(+\),\(-\),\(\times\),\(\div\)), then go to comparison operators (\(=\),\(\ne\),\(\geq\),\(\leq\)) and finally logical operators (and, or).

Arithmetic operators

The 4 base arithmetic operators are addition (+), subtraction (-), multiplication (*) and division (/). The following are some base examples of how these can be used.

Code 
```{r}
###| Addition
3+10

###| Subtraction
6-2

###| Multiplication
5*15

###| Division
23/7
```
[1] 13
[1] 4
[1] 75
[1] 3.285714
Note

It is important to note that R follows the traditional BODMASS order of operations. As such, we can use brackets to control the order in which operations are performed.

Code 
```{r}
###| BODMASS
(1+7)/2
1 + 7/2
3*5+2/3
3*(5+2)/3
```
[1] 4
[1] 4.5
[1] 15.66667
[1] 7

We can also do exponents (to the power of) using ^ or **, and we can find the modulo using %% and %/%

Code 
```{r}
###| Exponents 
5^2
5**2

4^(1/2)
4**0.5
```
[1] 25
[1] 25
[1] 2
[1] 2
Code 
```{r}
###| Modulo
10 %% 2 # remainder of 10 divided by 2
10 %% 4

10 %/% 2 # how many times 2 goes into 10
10 %/% 4
```
[1] 0
[1] 2
[1] 5
[1] 2

Comparison operators

Comparison operators compare 2 values to one another and will return TRUE or FALSE.

The 2 operators that can be used for any data type are equal to == and not equal to !=.

We can also compare two numbers using less than <, less than or equal to <=, greater than >, and greater than or equal to >=.

Code 
```{r}
###| Equal to
"a banana" == "a banana"
"a banana" == "a fruit"

7 == 5
5 == 5

###| Not wqual to
"a banana" != "a banana"
"a banana" != "a fruit"

7 != 5
5 != 5
```
[1] TRUE
[1] FALSE

[1] FALSE
[1] TRUE
[1] TRUE
[1] FALSE

```{r}
###| Less than

5 > 7 
7 > 5
5 > 5 # note that this will return FALSE!

###| Less than or equal to
5 <= 7 
7 <= 5
5 <= 5 # note that this will return TRUE!

###| Greater than or equal to
5 >= 7 
7 >= 5
5 >= 5 # note that this will return TRUE!
```

Logical operators

Logical operators allow us to combine things or negate something.

If we want to find the negative, we use the ! operator. For example, we may want to say something is not true. we do this with !TRUE. Or we may want to check if 5 is not less than 7, !(5<7).

Code 
```{r}
###| Negation of TRUE FALSE
TRUE
!(TRUE)

FALSE
!(TRUE)
```
[1] TRUE
[1] FALSE
[1] FALSE
[1] FALSE
Code 
```{r}
###| Example of negation on relational operators
5<7
!(5<7)

5==7
!(5==7)
```
[1] TRUE
[1] FALSE
[1] FALSE
[1] TRUE

We can also combine operators using AND, &. For example, we may want to check that 5<7 and 10<12. Alternatively, we may want to allow A or B to be true. We do this with |.

Code 
```{r}
###| And on TRUE FALSE
TRUE & TRUE
FALSE & FALSE
TRUE & FALSE

###| And on relational operators example
(7>5) & (10<12)
(7<5) & (10>12)
(7<5) & (10<12)
```
[1] TRUE
[1] FALSE
[1] FALSE
[1] TRUE
[1] FALSE
[1] FALSE
Code 
```{r}
###| OR on TRUE FALSE
TRUE | TRUE
FALSE | FALSE
TRUE | FALSE

###| OR on relational operators example
(7>5) | (10<12)
(7<5) | (10>12)
(7<5) | (10<12)
```
[1] TRUE
[1] FALSE
[1] TRUE
[1] TRUE
[1] FALSE
[1] TRUE

# Logical operators

Logical operators allow us to combine things or negate something.

If we want to find the *negative*, we use the `!` operator. For example, we may want to say something is not true. we do this with `!TRUE`. Or we may want to check if 5 is not less than 7, `!(5<7)`. 

```{r}
###| Negation of TRUE FALSE
TRUE
!(TRUE)

FALSE
!(TRUE)
```

```{r}
###| Example of negation on relational operators
5<7
!(5<7)

5==7
!(5==7)
```

We can also combine operators using AND, `&`. For example, we may want to check that `5<7` and `10<12`. Alternatively, we may want to allow A or B to be true. We do this with `|`. 

```{r}
###| And on TRUE FALSE
TRUE & TRUE
FALSE & FALSE
TRUE & FALSE

###| And on relational operators example
(7>5) & (10<12)
(7<5) & (10>12)
(7<5) & (10<12)
```

```{r}
###| OR on TRUE FALSE
TRUE | TRUE
FALSE | FALSE
TRUE | FALSE

###| OR on relational operators example
(7>5) | (10<12)
(7<5) | (10>12)
(7<5) | (10<12)
```
